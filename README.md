# student-management-system
A console-based application to manage student records.

## Features
-Add Student
-View all students
-Search by ID
-Update Student by ID
-Delete Student by ID

## Technologies used
Java
ArrayList
OOP concepts
Loops and Switch statements

## How It Works

The program displays a menu where the user can:
1. Add student details
2. View all stored students
3. Search for a student using ID
4. Delete a student using ID
5. Exit the program
